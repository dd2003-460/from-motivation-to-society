# Part 3 执行指导：AI 常见错误与正确意图说明

> 本文件供指导 AI（如 OpenClaw / Claude / GPT）执行 Part 3 写作任务时使用。
> 请将本文件作为系统提示前置，配合 `d_declaim_ai_system.md` 一同加载。

---

## 一、Part 3 的核心目标（一句话版）

**用"任何人都能跟上的自然逻辑链"，从最朴素的问题出发，逐步推导出经济、文化、政治、法律四个领域的统一视角——并在推导结束时，读者应感到"这和我的直觉完全一致，只是我从未系统说出来"。**

这不是后验解释，而是**前向推导**：给普通读者一把锁，告诉他钥匙在哪里，然后让他自己拧开门。

## 二、AI 当前执行的核心问题（已犯错误清单）

### 错误 1：把"统一视角"写成了"并列分述"

**错误做法**：
> 第一章讲经济，第二章讲文化，第三章讲政治，第四章讲法律，每章有自己的 logicmap，然后各自收尾。

**问题所在**：
这是四个**独立科目的教科书**组合，不是统一视角的推导。四章之间没有内在必然的逻辑推进关系，只是"主题相关"地摆在一起。

**正确做法**：
每章必须是下一章的**前提**，不是"主题切换"而是"推导前进"。正确链条是：

```
文化（最宽泛的模因传播）
  ↓ 加入稀缺约束（资源有限）
经济（带有交换门槛的模因子集）
  ↓ 加入身份/偏好结构（谁和谁交换、怎么分配）
政治（经济交换网络的人情/权力层）
  ↓ 所有这些需要确定性和可追责性
法律（将动态博弈结果编码为固定规则）
```

每一步的"加入什么"就是那一章的核心论证。**不是切换话题，而是叠加约束。**

---

### 错误 2：把随机性 / 演化动力学当成"背景装饰"而不是"结构生成器"

**错误做法**：
> 提一句"社会系统是演化的"，然后用静态的逻辑工具描述结果。

**问题所在**：
演化动力学和随机性本身**是 Part 3 结构的来源**，不是背景音乐。

**正确做法**：
Part 3 的核心论点是：
1. 社会结构不是"设计出来的"，而是在随机扰动下经历选择压后**留存下来的稳态**。
2. 这个演化过程有两种截然不同的模式：
   - **有压扩增（选择压）**：资源有限，个体竞争，只有有效的结构被保留。→ 经济和政治属于此列（受守恒约束，零和性质）
   - **无压扩增**：没有资源约束，传播不消耗原体，有效即扩张。→ 文化模因属于此列（非零和，可以无限复制）

这两种模式**必须在 Part 3 开头明确建立**，之后经济/政治/文化的所有结论都是这两种模式的具体例子。

---

### 错误 3：经济、政治、文化的关系被写成"平行"而非"嵌套"

**错误做法**：
经济 ≠ 文化，政治 ≠ 经济，三者是平行的不同领域。

**正确意图（作者原话）**：

> 经济是特殊的文化模因——带有经济门槛（稀缺资源约束）的模因行为。  
> 经济的交换和政治的人情网络交换具有同构性质。  
> 因此：政治 ≈ 经济 + 身份/偏好加权，经济 ≈ 文化 + 稀缺约束。

这是**嵌套关系**，而不是平行关系：

```
文化模因（最一般集合）
  ⊃ 经济行为（加稀缺约束后的子集）
    ⊃ 政治交换（加身份/偏好权重后的子集）
```

**推导要求**：
- 必须先建立"文化模因"的一般性定义（信息片段，传播不减少原体）
- 然后证明"当传播需要消耗稀缺资源时，文化退化为经济"——这是经济章的导出方向
- 然后证明"当经济交换引入身份认同和权力结构时，经济退化为政治"——这是政治章的导出方向
- **不是三者独立建立，而是逐步限制条件**

---

### 错误 4：法律被写成"政治的附属品"或"单独的规则体系"

**正确意图**：

法律的核心地位远超目前的写法。正确定位是：

> **法律是整个社会科学定量方法的基础——条件概率作为因子贡献的确定框架。**

具体说：
- 经济学、政治学、社会学都面临"多因一果"的测量问题：A、B、C 共同导致了结果 D，各贡献多少？
- 法律通过"定责"发展出了一套解决方案：**用条件概率差（有 vs 无某行为时结果概率的变化量）来定义因子贡献**
- 这个框架不只是"法律条文的逻辑"，它实际上是**社会科学因果推断的统一基础**

写法要求：
- 法律章不是"法律是什么"，而是"法律解决了社科定量的一个根本困难"
- 条件概率框架要被明确提升为：**任何多因一果场景的责任/贡献分解方法**，无论是法庭、社会政策评估、还是历史归因

---

### 错误 5：逻辑链断裂——读者无法自己"问出来"

**错误做法**：
提出结论，然后用5个段落论证它。读者是被动接受的。

**正确做法**：
每个新概念必须是对上一个概念的"自然追问的答案"。标准是：

> **如果一个读者只读了前一段，问了一个"但是，为什么……"，那这一段必须是那个问题的回答。**

检验方法：把相邻两段拿出来，看能不能在它们之间插入一个"但是，为什么……"的问题，使得逻辑无缝衔接。如果不能，说明逻辑链断裂了，需要补一个过渡段。

## 三、Part 3 的正确逻辑链（完整版）

这是作者意图的完整推导树。AI 必须按照这个结构推进：

### 第一层：演化动力学的两种模式（Part 3 导论，必须先建立）

```
问：社会结构是怎么来的？
答：不是设计，是演化——随机变异 + 选择压 + 留存
问：演化一定产生同样的结构吗？
答：不一定，要看"传播/扩张消不消耗稀缺资源"
  → 有压扩增（消耗）：竞争，零和，选择，收敛
  → 无压扩增（不消耗）：扩散，非零和，无限复制，多样化
问：这两种模式跟社会现象有什么对应？
答：——这就是接下来四章要推导的
```

### 第二层：文化模因（无压扩增的典型）

```
问：什么样的信息会自发传播？
答：触发情绪的、降低认知成本的、提供身份认同的
问：传播的时候消耗原体吗？
答：不消耗——说出一个故事，故事还在我这里
问：那传播的上限在哪里？
答：注意力和认知带宽（接收方资源），不是原体本身
→ 结论：文化遵循无压扩增逻辑，不守恒，爆炸性增长
问：那文化的"关键指标"是什么？
答：传播强度（intensity），而不是方向（direction）
  → 情绪激活度决定传播力，内容正确与否是次要的
  → 这解释了为什么谣言、煽动性内容比事实更快传播
```

### 第三层：经济（文化 + 稀缺约束 → 有压扩增）

```
问：如果一种文化模因的传播需要消耗稀缺资源怎么办？
答：就不再是无压扩增了——变成了有压扩增
问：什么样的模因需要消耗资源？
答：那些包含"交换"行为的模因——你要得到某物，必须付出某物
问：这不就是经济行为吗？
答：对。所以经济是"带有稀缺约束的文化模因子集"
问：稀缺约束带来了什么新性质？
答：守恒性——你多得我就少得，零和博弈开始起作用
  → 经济网络在总量约束下具有守恒性质（价值不凭空创造/消灭）
问：供求关系从哪里来？
答：稀缺产生稀缺端（供方），需求来自模因层面的偏好
  → 价格是稀缺信号，不完全信息导致价格信号失真
  → 追涨杀跌是不完全信息下的理性启发式（模仿他人作为信号处理）
```

### 第四层：政治（经济 + 身份/偏好权重 → 权力网络）

```
问：经济交换是对称的吗（双方地位相等）？
答：理论上是，实际上不是——因为交换者有身份、历史、偏好
问：当交换双方的身份关系变得重要，会发生什么？
答：交换不再只是"等价物品"，而是"人情、义务、权力的互换"
  → 政治从经济中分离出来的那一刻，是"交换中加入了不可货币化的偏好"
问：政治和经济的同构性在哪里？
答：都是交换网络；货币 ↔ 权力/人情；定价 ↔ 权力赋予/撤销；市场 ↔ 政治博弈场
问：为什么政治不遵循严格守恒？
答：因为权力的"赋予"来自认同（对职能的集体认同），认同可以凭空增加或消失
  → 政治守恒性弱于经济，但仍受资源约束（军队、税收、信息控制）
问：科层制从哪里来？
答：个人权力（有压扩增，随人消亡）的信息损失问题
  → 把个人魅力"编码"为职能程序，就是机构化——等同于文化模因化
```

### 第五层：法律（将博弈结果编码为可追责规则 + 因果贡献的定量方法）

```
问：动态博弈产生了结果，怎么确定谁该负责？
答：用条件概率——P(结果 | 有行为A) vs P(结果 | 无行为A)
问：这个思路可以推广吗？
答：可以——这正是社会科学"因果推断"的核心问题：多个因素，各贡献多少？
问：为什么法律发展出了这套工具？
答：因为法律是唯一被强制要求"给出确定性答案"的系统
  → 不能说"有点像他的错"，必须说"70%他的责任"
  → 这个压力迫使法律发展出最精细的因果分解工具
问：这个工具意味着什么？
答：条件概率框架可以统一所有社科的因子贡献问题：
  经济学的处置效应 ≈ P(结果|处置) - P(结果|无处置)
  政治学的政策效果 ≈ 相同条件概率差
  历史归因的"必然性" ≈ 沿因果链的条件概率乘积
  → 法律不只是规则体系，而是社科定量方法的基础逻辑
```

## 四、章节写作的执行规则

### 规则 1：每一章的开头必须从"上一章遗留的问题"开始

不能说"本章讨论政治"。必须说：
> "上一章我们看到经济交换在身份和偏好介入后开始变形——但这个'偏好'到底从哪里来，又如何稳定化？这就是政治诞生的问题。"

### 规则 2：每个新概念必须通过"加入一个约束"导出，不得凭空引入

正确示例：
> "如果文化模因的传播不需要消耗原体（你把想法告诉我，你还有这个想法），那它的扩张是无限的。但现在加入一个约束：传播需要消耗稀缺资源。这一步改变了所有性质——这就是经济行为从文化行为中分离出来的时刻。"

错误示例：
> "经济的本质是供求关系……"（直接跳入结论，未从前章推导）

### 规则 3：守恒性 vs 无压扩增，必须在文化章和经济章之间写清楚

这是全书 Part 3 最重要的结构分叉点。不能一笔带过，必须用完整的一节（subsection）论证：
- 文化模因：传播是复制（A→A+B），不消耗 A，总量可以无限增长
- 经济交换：交换是转移（A→B），A 消失了，总量守恒
- 政治权力：赋予/撤销权力是认同行为，部分守恒（有资源基础），部分不守恒（认同可以消失）

### 规则 4：法律章必须明确指出它的定量意义

不要满足于"法律区分故意和过失"这类描述性写作。必须推进到：
> "这整套体系的底层是条件概率框架。一旦你接受'责任∝P(损害|行为)-P(损害|无行为)'，你就得到了社会科学因果推断的统一语言。这不是法律的特殊发明，而是它被迫解决了其他社会科学一直在回避的问题：在多因素介入的真实世界里，如何仍然给出一个数字。"

### 规则 5：统一视角的表述必须出现在 Part 3 结尾

Part 3 最后必须有一节叫做"统一视角"或"三者的关系"，明确写出：

```
文化  ⊃  经济  ⊃  政治（嵌套关系，逐步限制）
  ↓
法律（为三者的动态博弈结果提供定量追责框架）
  ↓
这四者不是四个领域，而是同一个"认同+资源+传播"动力学在不同约束下的同一张脸
```

## 五、AI 的常见走偏与纠正方式

| AI 走偏行为 | 纠正指令 |
|---|---|
| 把四章写成独立教科书 | "重新写，每章第一段必须明确承接上章的遗留问题" |
| 随机性/演化论只被提及不被用 | "在 Part 3 导论里增加一节：有压 vs 无压扩增，此后每章开头标注'本章是哪种扩增模式'" |
| 经济、文化、政治被平行处理 | "重构关系：文化是大集合，经济是加稀缺约束的子集，政治是加偏好权重的子子集。必须用集合嵌套的逻辑表达" |
| 法律只讲规则，不讲定量意义 | "法律章的结尾必须有一节：'条件概率框架作为社科定量的统一基础'，明确与统计/计量的因果推断对接" |
| 逻辑链断裂（结论突然出现） | "在结论段和前段之间插入一个问题：'但这时读者自然会问……'，然后用这一段回答那个问题" |
| 用中英混杂但不标注 | "所有英文术语必须格式为：中文（English），例如：信息级联（information cascade）" |

## 六、检验标准：Part 3 完成后的自测问题

在提交给作者之前，AI 必须能回答以下问题。如果任何一个回答需要超过两句话，说明写作没有达成目标：

1. **文化和经济的区别是什么，用一句话说？**
   > 正确答：传播行为是否消耗稀缺资源——不消耗是文化，消耗是经济。

2. **政治是从经济中如何导出的，用一句话说？**
   > 正确答：当经济交换引入不可货币化的身份偏好，交换网络就具有了权力层——这就是政治。

3. **条件概率和法律的关系是什么，用一句话说？**
   > 正确答：法律定责本质上是计算P(损害|行为) - P(损害|无行为)，这个框架统一了所有社科的因果贡献分解问题。

4. **法律、经济、政治、文化之间最简洁的关系是什么？**
   > 正确答：文化⊃经济⊃政治（按稀缺约束和偏好权重嵌套），法律为三者的博弈结果提供条件概率追责框架。

5. **普通人如果从头问"为什么"，能不能自己推导到这些结论？**
   > 验证方法：给一个没读过本书的朋友看 Part 3 的第一节，看他能否自己问出"但是如果加上资源约束呢"这类问题。如果需要他先接受某个未推导的概念，就说明写法有问题。

## 七、关于介观统一论（暂不写入正文，写入附录/论文）

作者已明确：介观统一论（经济=模因压缩，政治=带偏好经济，三者统一）的严格证明留给论文，正文只需要呈现结论性视角。

**正文的处理方式**：
- 在 Part 3 结尾的"统一视角"节中，陈述三个命题（不加严格证明）：
  1. 经济是稀缺化的模因
  2. 政治是人情化的经济
  3. 法律是博弈结果的条件概率编码
- 加注：
  > "以上三个命题的严格推导见附录/待发表论文《介观统一论》。此处只作为统一视角的直觉呈现。"

**不要让 AI 去"证明"这些命题**，AI 会产生虚假的推导。只让它**描述**，并**指向附录**。
